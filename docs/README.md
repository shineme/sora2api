## 🛠️ 技术栈信息

### 后端技术
- **框架**: FastAPI 0.119.0
- **语言**: Python 3.8+
- **数据库**: SQLite + aiosqlite
- **认证**: JWT + bcrypt
- **配置**: TOML格式

### 前端技术
- **页面**: 原生HTML
- **样式**: TailwindCSS
- **脚本**: Vanilla JavaScript
- **图标**: SVG图标

### 部署方案
- **容器化**: Docker + Docker Compose
- **服务器**: Uvicorn (ASGI)
- **代理**: 支持HTTP/HTTPS/SOCKS5
